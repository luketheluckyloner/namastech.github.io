Resumen

El USB Rubber Ducky se ve como una memoria USB común para humanos, pero para el computador es un teclado HID que inyecta payloads a velocidades sobrehumanas. Modernizado con USB-C y USB-A, usa DuckyScript™ 3.0, Payload Studio y PayloadHub para crear, compartir y ejecutar ataques de hotplug creativos, ideal para profesionales de ciberseguridad y educación.
