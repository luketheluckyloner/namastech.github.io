Resumen

El Wi-Fi Nugget es un microcontrolador compacto basado en ESP8266, diseñado para aprendizaje práctico y pruebas de seguridad inalámbrica. Viene completamente ensamblado y probado: incluye pantalla SH1106, botones, headers y la placa D1 mini, listo para usar con Arduino IDE o MicroPython. El dispositivo se entrega preinstalado con Nugget Deauther Firmware para arrancar rápido en proyectos educativos.
